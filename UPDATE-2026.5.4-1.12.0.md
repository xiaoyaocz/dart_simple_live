# UPDATE

## 2026-05-03

### Fix

- 弹幕聊天用户名和内容统一做了左右 `trim`，并修复了用户名过长时单独换行的问题。
- 修复虎牙头条 / 置顶弹幕（SC）无法显示、价格异常、内容为空仍尝试展示的问题。
- 修复虎牙头条 / SC 高频刷新时的重复堆叠问题，补充了更稳定的去重与更新逻辑。
- 修复 B 站贡献榜重复刷新、重复更新、刷新异常的问题，改为按需拉取。
- 修复 Windows 端小窗、全屏、窗口恢复、最大化切换后的显示异常问题。
- 修复 Windows 端小窗退出后窗口置顶状态残留的问题。
- 修复 SC 倒计时在数据刷新后不同步的问题。
- 修复 SC 卡片销毁时定时器空引用的风险。
- 修复旧工作流把 Android 与 macOS 绑定在同一个 job 中，导致 APK 已构建成功但整次 workflow 仍显示失败的问题。
- 修复 Windows 发布仍然带出 `msix` 的问题，正式发布改为只保留 `zip`。

### Optimize

- 对直播间信息、用户名、消息文本、SC 内容做统一清洗，减少空格和脏数据带来的显示问题。
- 为 SC 增加更稳定的唯一标识与指纹去重逻辑，降低重复插入概率。
- 优化虎牙 SC 的刷新频率和状态清理逻辑，减少旧数据残留。
- 为弹幕重建增加回放机制，窗口模式切换后可以更平滑地恢复视频区弹幕。
- 贡献榜支持懒加载后的首次自动读取、手动刷新和更清晰的分层展示。
- 正式 Release 资产命名和产物归档规则统一。

### Update

- 新增弹幕屏蔽总开关，不需要删除屏蔽词也能一键临时关闭屏蔽。
- 新增关键词屏蔽独立开关和用户屏蔽独立开关。
- 新增用户屏蔽列表，关键词和用户屏蔽现在分开管理。
- 新增直播间内点击用户名快速屏蔽 / 取消屏蔽，长按用户名可复制。
- 新增屏蔽预设功能，支持启用与删除。
- 新增 `B站 / 抖音 / 斗鱼` 的直播间贡献榜或亲密榜抓取与展示能力。
- GitHub Actions 已拆分为 3 条独立自动发布工作流：Android、Windows、Linux。
- 新增 `macOS` 手动构建工作流入口，且不再阻塞正式发布。

## 2026-05-04

### Fix

- 修正了直播间内“正在连接弹幕服务器”“线路选择”“线路 N”等中文文案乱码。
- 直播间标题改为按内容区单独居中，不再被右侧操作按钮宽度带偏。
- 虎牙直播间详情补回分区字段，抖音分区反查逻辑补强，同类推荐可用性提升。
- 分类页在分区缺少原始图标时改为显示统一的占位图标，抖音分类观感更统一。
- Windows 端在一段时间无操作后，鼠标箭头会自动隐藏。
- 安卓端弹幕、直播间和后台恢复逻辑已补强，应用被系统杀掉后可继续恢复上次房间。

### Optimize

- 弹幕显示从“上下间距”改成“显示几行”控制，并根据显示区域、字体大小动态计算可显示行数。
- 弹幕过稀时会在设置页下方提示：`你选的行数太少了！我心里空落落的~`
- 弹幕延迟改为可按平台单独微调。
- 贡献榜增加了前十、粉丝牌、高贡献筛选。
- 直播间补充了观看历史、同类推荐和快捷入口。
- 直播间用户点击操作继续补强：屏蔽、临时禁言、备注、复制、批量恢复。

### Update

- 弹幕屏蔽改成分平台管理，支持关键词和用户分别一键清空，并增加确认对话框。
- 屏蔽预设补上了导入、导出和编辑保存覆盖能力。
- 贡献榜增加了开关控制，避免不需要的平台重复拉取。
- 抖音分区信息补齐了标签图标和房间分区字段。

### Release 补充（2026-05-04）

- 修复 Windows 端全屏播放器底部“清晰度 / 线路”等功能区在鼠标悬浮时持续闪动的问题。
- 调整 Windows 端直播间 AppBar 标题居中逻辑，按主内容区计算视觉中心，不再把右侧整列功能栏宽度计入居中范围。
- 虎牙 / B站直播间在没有头条或 SC 时，补充“当前直播间无头条内容 / 当前直播间无 SC 内容”空状态提示。
- 修复安卓端全屏观看直播时，长按后进入快捷入口里的“观看历史”显示异常问题，统一改为走移动端菜单路径。
- 直播间内的关注列表补齐与主页一致的筛选能力，支持“全部 / 直播中 / 未开播”切换。
- 增强安卓端后台返回后的直播恢复逻辑，并补充 `WAKE_LOCK` 权限，降低长时间挂后台后播放被中断的概率。
