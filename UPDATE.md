# UPDATE

## 2026-05-03

### Fix

- 弹幕聊天用户名和内容统一做了左右 `trim`，并修复了用户名过长时单独换行的问题。
- 弹幕屏蔽改成分平台管理，支持关键词和用户分别一键清空，并增加了确认对话框。
- 屏蔽预设补上了导入、导出和编辑保存覆盖能力。
- 安卓端弹幕、直播间和后台恢复逻辑已补强，应用被系统杀掉后可继续恢复上次房间。
- 贡献榜增加了开关控制，避免不需要的平台重复拉取。
- B 站贡献榜刷新/重复更新问题已处理，改为只在需要时拉取。
- 直播间用户点击操作已补全：屏蔽、临时禁言、备注、复制、批量恢复。
- 修复了Windows端小窗后全屏导致窗口大小错误的问题。

### Optimize

- 弹幕显示从“上下间距”改成“显示几行”控制，并根据显示区域、字体大小动态计算可显示行数。
- 弹幕过稀时会在设置页下方提示：`你选的行数太少了！我心里空落落的~`
- 弹幕延迟改为可按平台单独微调。
- 贡献榜增加了前十、粉丝牌、高贡献筛选。
- 直播间补充了观看历史、同类推荐和快捷入口。
- 抖音分区信息补齐了标签图标和房间分区字段。

## 2026-05-04

### Fix

- 修正了直播间内“正在连接弹幕服务器”“线路选择”“线路 N”等中文文案乱码。
- 直播间标题改为按内容区单独居中，不再被右侧操作按钮宽度带偏。
- 虎牙直播间详情补回分区字段，抖音分区反查逻辑补强，同类推荐可用性提升。
- 分类页在分区缺少原始图标时改为显示统一的占位图标，抖音分类观感更统一。
- Windows 端在一段时间无操作后，鼠标箭头会自动隐藏。
